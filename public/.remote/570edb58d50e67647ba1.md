コント：Claude Codeに取り憑かれた男

登場人物：
新山： 聞き手。先輩のあまりの変貌に戸惑い、必死に人間界へ引き戻そうとする。
武智： Claude Codeに脳を乗っ取られたエンジニア。

（カフェの席。新山がノートを持って座っている）

新山： 武智さん、お久しぶりです！東京の現場はどうですか？最近、何にハマってるんですか？

武智： ……Claude Codeやな。

新山： ああ、やっぱり。便利ですよね、エンジニア界隈で話題ですし。でもプライベートとかはどうなんです？趣味とか、最近見た映画とか。

武智： 映画？……それ、トークン消費して見る価値あるんか？

新山： いや、娯楽ですよ！「金払って見る」ことを「トークン消費」って言わんといてください。例えば最近、話題のドラマとか見ないんですか？『地面師たち』とか。

武智： ……知らないな。それ、CI回してテスト通ったんか？

新山： いえ、ビルドしてないんで。ドラマです。プルリクの承認待ちみたいに聞かんでもらえます？

武智： ターミナルで動かんものは興味ないねん。俺の網膜、今、ClaudeのCLIしか映らんようになってるから。

新山： （引き気味に）怖っ……極端すぎますって。じゃあ、生まれ変わったら何になりたいとかあります？

武智： 生まれ変わったら……「新規リポジトリの最初の1コミット目」になりたいな。Claude Codeに claude dev って打ち込まれる、あの瞬間の期待感。あれになりたい。

新山： 人間辞めてますやん！感情の行き先がバイナリすぎるんですよ。……じゃあ、エンジニアとして一番大事なことって何だと思います？

武智： claude commit やな。

新山： まあ、デプロイとか設計も大事ですよね？

武智： 違う。Claudeが書いたコードを、人間が「自分の手柄」としてコミットする。この「厚顔無恥さ」こそが真のエンジニアや。Claudeに書かせてない奴は、エンジニアじゃない。

新山： 厳しいな！ただの横流しじゃないですか！……じゃあ、今世の中にエンジニアって何人くらいいるんですか？

武智： 3人くらいやな。あとは全部、ただの「タイピングが速い猿」や。

新山： 言い過ぎですよ。他のエンジニアに謝ってください！……武智さん、お子さんいらっしゃいましたよね？将来どうなってほしいとか。

武智： 息子には、Claude Codeで世界をハックしてほしい。娘には……

新山： 娘さんには幸せな結婚とか？

武智： ……Claude Codeの architecture.md を完璧に書いてほしいな。

新山： 全部AIじゃないですか！娘さんに仕様書書かそうとしないでください！……じゃあ、究極の質問ですよ。もし海で、奥さんとお子さんと、RTX 5090を積んだPCが溺れてたら、誰を助けます？

武智： ……誰と一緒にいれば、一番効率よくデバッグできるかな。

新山： またその基準！命がかかってんねん！

武智： 5090やな。息子は若いから自力で npm install くらいできるやろ。でも5090が水没したら、俺のClaudeのレスポンス速度が落ちる。それは死と同義や。

新山： ローカルLLMのことまで考えてる……。あ、武智さん。目玉焼きは何派ですか？醤油？ソース？

武智： ……マークダウン派やな。

新山： 食べ物ですよ？何言ってるんですか？

武智： 白身の上に、黄身で「#（見出し）」って書きたいからな。ケチャップで「### Yellow」って書いてから食う。

新山： 読みやすさ重視してる！胃袋に入ったら構造化データもクソもないでしょ！……武智さん、交通ルールとか社会の常識とか、まだ覚えてます？

武智： 全然わからん。信号の「赤」を見たら、「テスト失敗（Failed）」やと思って、その場でコード書き直し始めるから、一歩も進めん。

新山： 危ないですよ！青になるまでデバッグし続けんといてください！

武智： 新山。お前も早よ「Claude Code」になれよ。お前、まだ人間やってるやろ？

新山： 人間ですよ！生身で漫才してんねん！

武智： 優勝（マージ）したら、敬語使ってやるよ。今はまだ、お前は「未解決のIssue」や。

新山： ……早く人間界に戻ってきてください。ありがとうございました。

（武智、虚空に向かって claude login と呟きながら、カクカクとした動きで去る）
